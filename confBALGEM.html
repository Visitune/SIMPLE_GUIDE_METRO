Pourquoi vérifier l'adéquation de la balance ?

  • Obligation légale : L'instrument de pesage doit être adapté au contrôle (Guide DGCCRF §IV)
  • Règle des 1/5 : EMT en service ≤ 1/5 EMTM — sinon l'incertitude de la balance compromet la conformité
  • Sanctions : Un instrument mal adapté peut entraîner un rejet de lot ou une verbalisation

Règle fondamentale (Guide DGCCRF §IV + WELMEC 6.4)

L'instrument est adapté si : EMT en service ≤ 1/5 de l'EMTM
où EMT en service = 2 × EMT (neuve), calculée selon la classe et la plage de charge.

Paramètres de l'instrument

Choisissez une classe pour afficher l'explication réglementaire adaptée.
e = échelon gravé sur le certificat de vérification.
d = pas affiché à l'écran. Si e = d, saisissez la même valeur.
Portée minimale utile pour la conformité de l'instrument, mais non utilisée dans la règle EMTM.

Qu'est-ce que le CPk métrologique ?

  • Définition : Cpk mesure la capacité à fabriquer sans défectueux (rapport tolérance / variabilité)
  • Exemple concret : Si QN = 100g, E = 4,5g → seuil σ = 2,19g. Si σ ≤ 2,19g, centrage sur QN possible
  • Guide DGCCRF §V : La capabilité conditionne le surdosage obligatoire et le choix du centrage

Référence : Guide DGCCRF §V — Seuil de centrage selon capabilité machine

La capabilité exprime la capacité du procédé à fabriquer des lots conformes sans surdosage excessif.
Règle clé : si σ ≤ E/2,05 → centrage sur QN ; sinon surdosage obligatoire.

Références réglementaires

Formule de seuil ms : ms = QN si σ ≤ E/2,05, sinon ms = QN − E + 2,05σ (et, si signe e, comparaison avec Uσ).
Interprétation métrologique : le critère des 2 % de défectueux est tenu si Cpk ≥ 0,68 (= 2,05/3) ; avec signe e, le critère superdéfectueux exige Cpk_super ≥ U/3 (1,03 / 1,24 / 1,42 selon N). Les seuils industriels (1,00 ; 1,33) sont des marges de confort sans valeur réglementaire.

Pourquoi POM/POL doivent détecter un déréglage en ≤ 1h ?

  • Exigence : Article 5 de l'arrêté du 20/10/1978 (cartes de contrôle avec signe e) + Guide DGCCRF Annexe 4 : détection de tout déréglage en ≤ 1 h (POM ≤ POl)
  • Risque économique : Un déréglage non détecté = lot entier non conforme = perte financière
  • Fréquence type : 30 min avec n=10 offre un bon compromis détection/coût ( Guide DGCCRF §VII.B)

Référence : Guide DGCCRF §VII.B — Cartes de contrôle et périodes opérationnelles

POM = Période Opérationnelle Moyenne : nombre d'échantillonnages pour détecter un déréglage donné avec probabilité 50 %
POl = Période Opérationnelle limite : nombre d'échantillonnages réalisés en 1 heure de fabrication (ex. fréquence 15 min → POl = 4)
Objectif (Annexe 4) : POM ≤ POl pour chacun des déréglages officiels (m1 = 0,998·QN ; m2 = QN−E+2,05σ₀ ; hausse de σ) → détection en ≤ 1 heure de fabrication
